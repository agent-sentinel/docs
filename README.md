# Agent Sentinel Docs (Mintlify)

This directory contains the Agent Sentinel documentation site, powered by Mintlify.

## Preview locally

```bash
npm i -g mint
cd /home/jimmy/development/agent-sentinel/docs
mint dev
```

Then open `http://localhost:3000`.

## API reference source

The platform API reference is generated from:
