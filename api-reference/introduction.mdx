
## Authentication

Most endpoints require an `Authorization` header:

```bash
Authorization: Bearer <token>
```

For ingestion, the platform accepts either:

- **JWT access token** (obtained via `POST /api/v1/login/access-token`)
- **API key** (created via `POST /api/v1/api-keys/`)

<Tip>
  For agents, prefer **API keys** over user JWTs.
</Tip>
